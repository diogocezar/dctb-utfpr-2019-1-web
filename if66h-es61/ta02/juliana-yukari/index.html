HTML - Hypertext Markup Language

O HTML é uma das linguagens que utilizamos para desenvolver websites.
Seu acrônimo HTML vem do inglês e significa Hypertext Markup Language.
É a linguagem que irá exibir a informação.

CSS

O Cascading Style Sheets (CSS) é uma linguagem utilizada para definir a apresentação (aparência) de documentos que adotam para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML e etc..). O CSS define como serão exibidos os elementos contidos no código de um documento e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento. Ou seja, adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.
